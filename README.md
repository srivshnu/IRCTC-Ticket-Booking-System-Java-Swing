# IRCTC-Ticket-Booking-System-Java-Swing
Desktop app gui simulating IRCTC railway ticket booking: user login/register, train search by route/date, seat availability, multi-passenger booking, fare calculation, payment flow, and personal history. Built with Core Java, Swing GUI, OOP design (User/Train/Passenger/Booking classes), and CardLayout navigation. In-memory data for learning focus
